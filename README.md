# Perks

A simple Perk System for Pocketmine.

## Features
+ [x] No Hunger Perk
+ [x] Fly Perk
+ [x] Doublehealth Perk
+ [x] Autosmelt Perk
+ [ ] No Falldamage Perk
+ [ ] Nightvision Perk
+ [ ] Luck Perk
+ [ ] Speed Perk
+ [ ] Fire Resistance Perk
+ [ ] Haste Perk
+ [x] UI for the perks
+ [ ] UI for buying perks
+ [x] EconomyAPI Support
+ [x] BedrockCoins Support
+ [ ] Languages
+ [x] JSON Support
+ [ ] YAML Support
+ [ ] MySQL Support
